# World Awakened Example Pack

Optional example datapack content for the World Awakened framework.

This pack is intentionally external to the mod jar so the framework does nothing until a user chooses to install a datapack.

## Install

Copy `worldawakened_example_pack` into a world's `datapacks/` folder, then run:

```text
/reload
/wa reload validate
```

## Contents

- example mutation definitions and pools
- example ascension rewards and offers
- example rule chains for mutator pressure and ascension offer cycles

These files are examples and starter content, not mandatory runtime content.
